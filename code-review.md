## Summary

This code review examined the original weight tracking Android app from CS 360 and identified planned improvements across software design and engineering, algorithms and data structures, and databases.

The review focused on existing functionality, code structure, incomplete features, data handling, and security concerns that became the basis for the three enhancements.

## Planned Enhancements

### Enhancement One: Software Design and Engineering

The first enhancement focused on cleaning up the app structure, separating adapter classes, completing the goal feature, improving SMS navigation, and making the app easier to use.

### Enhancement Two: Algorithms and Data Structures

The second enhancement focused on making better use of stored weight data by adding date-range filtering, a trend summary, sorting, real-time search, and calendar entry counts.

### Enhancement Three: Databases

The third enhancement focused on the database layer by hashing passwords, moving goal storage into SQLite, adding an index to support date filtering, and rewriting database upgrade logic to preserve existing weight entries.
